# PP Editorial New Ultralight Font Setup

To use the PP Editorial New Ultralight font in this project:

1. Purchase or obtain the font from its official source (Pangram Pangram Foundry)
2. Place the font file in: `/public/fonts/PPEditorialNew-Ultralight.woff2`

The code is already set up to use this font. If the font file isn't available, 
the text will fall back to the system's serif font.

## Font Information
- Font name: PP Editorial New Ultralight
- Font foundry: Pangram Pangram Foundry
- Official website: https://pangrampangram.com/products/pp-editorial-new

## File Format
For best web performance, we're using the WOFF2 format. If you have the font in a different 
format like OTF or TTF, you can convert it to WOFF2 using online converters.
