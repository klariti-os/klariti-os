  Setup flow (2 steps):
  1. Register NFC tag — Big NFC icon, "Scan NFC Tag" button writes a UUID token
  to the tag. Once registered shows a checkmark + "Continue" button
  2. Choose apps to block — App picker with FamilyControls. "Done" button
  completes setup

  Home screen (unlocked):
  - Lock open icon + "Ready to focus"
  - Shows how many apps will be blocked
  - "Start Focus" button (disabled until apps are selected)
  - "Edit blocked apps" link + menu for re-registering NFC or resetting

  Locked screen:
  - Full screen, prominent lock icon + "Focus Mode" title
  - "Tap to Unlock" button triggers NFC scan — only the registered tag's payload
   matches

  Under the hood:
  - activitySelection (FamilyActivitySelection) is JSON-encoded to UserDefaults
  so shields can be re-applied on app relaunch while locked
  - Shields persist in ManagedSettingsStore even after the app closes
  - NFC error code 200 (user cancelled) is silently ignored — no spurious error
  alerts
  - All state persisted with @AppStorage — correct screen shown on every launch
