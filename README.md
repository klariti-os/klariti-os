# Implementation for chromium-based browser

- first time creating an extension ? [Follow the docs from google](https://developer.chrome.com/docs/extensions/get-started?hl=de)

# How to run example extension

- this extension quickly takes you to the klariti-os official landing page upon being clicked
- save the /chromium folder to your machine
- go to chrome://extensions and toggle "developer mode" on
- click "load unpacked" and select the saved chromium folder
- the new extension should now be loaded! Click the extentions icon in the top right to pin the new extension
- clicking its icon will open https://klariti-os.vercel.app/ on a new page

# Extention for chromium-based browser to prevent access to instagram.com

- popup page With toggle ON/OFF switch
- when toggled ON, any open tabs with instagram.com will be closed, an tabs opened to instagram.com will be closed

# How to run example instagram blocking extension

- save the /instagram-popup folder to your machine
- go to chrome://extensions and toggle "developer mode" on
- click "load unpacked" and select the saved /instagram-popup folder
- the new extension should now be loaded! Click the extentions icon in the top right to pin the new extension
- toggle your broswer's acces to instagram.com
